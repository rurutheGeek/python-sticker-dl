# LINEスタンプ・絵文字 ダウンローダー

LINEストアのURLから、スタンプや絵文字をダウンロードし、整理・リネームするPythonスクリプトです。
ダウンロード手順は Manual_Download_Guide.md を参照。

## 機能

- LINEストアのURLを基に、スタンプや絵文字の情報を自動で取得します。
- 対話形式でダウンロードしたいコンテンツの種類（普通・メッセージ・動くスタンプ、普通・動く絵文字など）を選択できます。
- 選択された種類に応じて、適切なURLからZIPファイルをダウンロードします。
- ダウンロードしたZIPを解凍し、スタンプ・絵文字本体の画像ファイルのみを抽出します。
- 抽出した画像を「（パックのタイトル）\_（元のファイル名）」の形式でリネームします。
- 一時ファイルや不要なメタデータは自動で削除されます。

## 必要なパッケージ

このスクリプトを実行するには、以下のPythonパッケージが必要です。

- `requests`
- `beautifulsoup4`

これらのパッケージは`requirements.txt`に記載されています。

## インストール手順

`requirements.txt` を使って、必要なパッケージをインストールします。
```bash
uv pip install -r requirements.txt
```

## 使い方

スクリプトは2通りの方法で実行できます。

### 1. 引数でURLを渡す方法

コマンドの後ろに直接LINEストアのURLを指定します。

```bash
python main.py "https://store.line.me/stickershop/product/26959/ja"
```

### 2. 対話形式でURLを入力する方法

引数なしでスクリプトを実行すると、URLを入力するように求められます。

```bash
python main.py
```
実行すると、以下のようにURLの入力を求められます。
```
LINEストアのURLを入力してください: 
```

### 実行後の対話

URLを渡すと、スクリプトはスタンプの種類を尋ねてきます。該当する番号を入力してEnterキーを押してください。

**例（スタンプの場合）:**
```
スタンプの種類を選択してください (1: 普通, 2: メッセージ, 3: 動く): 3
```

**例（絵文字の場合）:**
```
絵文字の種類を選択してください (1: 普通, 2: 動く/音あり): 1
```

その後、ダウンロードと整理が自動的に開始されます。完了すると、`output` ディレクトリ内にスタンプのタイトル名のフォルダが作成され、その中にリネームされた画像ファイルが保存されます。
